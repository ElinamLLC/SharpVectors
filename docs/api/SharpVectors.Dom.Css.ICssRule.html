Interface ICssRule

Namespace
SharpVectors.Dom.Css
Assembly
SharpVectors.Core.dll

The ICssRule interface is the abstract base interface for any type of CSS statement. This includes both rule sets and at-rules. An implementation is expected to preserve all rules specified in a CSS style sheet, even if the rule is not recognized by the parser. Unrecognized rules are represented using the ICssUnknownRule interface.

public interface ICssRule

The type of the rule, as defined above. The expectation is that binding-specific casting methods can be used to cast down from an instance of the ICssRule interface to the specific derived interface implied by the type.

Type

If this rule is contained inside another rule (e.g. a style rule inside an @media block), this is the containing rule. If this rule is not nested inside any other rules, this returns null
