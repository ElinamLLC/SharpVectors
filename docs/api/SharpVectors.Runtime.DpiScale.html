    Remarks

    Two DPI scale values are equal if they are equal after rounding up to hundredths place.

    Common PPI values in use are:

    PPIDPI(%)DPI(Ratio)
    96100%1.00
    120125%1.25
    144150%1.50
    192200%2.00
