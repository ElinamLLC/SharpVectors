    Interface ISvgRenderer

    Defines the interface required by a renderer to render the SVG DOM.

    Namespace: SharpVectors.Dom.Svg
    Assembly: SharpVectors.Core.dll
    Syntax
    public interface ISvgRenderer
    Remarks

    The ISvgRenderer is used to render a ISvgElement object onto a bitmap. During the rendering process, it will also generate RenderingNode objects for each XmlElement object in the DOM tree to assist in the rendering.
