SvgTransformF(SvgRectF, SvgPointF[])

Initializes a new instance of the SvgTransformF class to the geometric transform defined by the specified rectangle and array of points.

public SvgTransformF(SvgRectF rect, SvgPointF[] plgpts)

Parameters

rect SvgRectF

A SvgRectF structure that represents the rectangle to be transformed.

plgpts SvgPointF[]

An array of three SvgPointF structures that represents the points of a parallelogram to which the upper-left, upper-right, and lower-left corners of the rectangle is to be transformed. The lower-right corner of the parallelogram is implied by the first three corners.

Exceptions

ArgumentNullException

If the plgpts is null.

ArgumentException

If the length of the plgpts array is not equal to 3.

ArgumentOutOfRangeException

If the width or height of the rect is zero.
